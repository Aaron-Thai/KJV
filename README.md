# Purpose
Read the KJV Bible online with a minimal webpage.  
Open on Microsoft Edge for Read Aloud options.

<b> Input format: </b> BookChapter:Verse
* John1 (for the whole chapter)
* John1:1 (for the first verse of the chapter)
